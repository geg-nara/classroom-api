# Classroom API 実例集

## 利用の手順

1. レポジトリをダウンロードして、試用するスクリプトがあるフォルダにカレントディレクトリを移動します。

```
$ cd classroom-api/python
```

2. OAuth でデスクトップのアカウントを `credentials.json` を作成します。

3. スクリプトを実行します。

```
$ python createCourses.py
```
